# A C++ Sudoku solver

A Sudoku solver, written in C++. Feel free to contribute and propose improvements.

The solver is written the Dancing Link version of the Algorithm X. 


## Technicalities

 * language: C++11
 * interfaces: one command line interface and one graphical interface \see cli.cpp qt.cpp
 * configuration: CMake
 * documentation: [Doxygen](https://www.doxygen.nl/index.html)
 * tests: [catch.hpp](https://github.com/catchorg/Catch2/)
