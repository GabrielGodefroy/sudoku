# Python3 sudoku solvers


A Sudoku solver, written in Python. Feel free to contribute and propose improvements.

Python is an efficient language for prototyping, so several algorithms are proposed.

## Technicalities

 * language: 		Python
 * interfaces: 		one command line interface and one graphical interface (PyQt)
 * configuration: 	conda environment
 * documentation: 	sphinx (todo)
 * tests: 			pytest
